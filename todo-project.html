Todo app for Flutter Udacity course

Collaborated with Google on this open-source mobile app used in the course material for the Udactiy course teaching Google's new Flutter technology. The course demonstrates how the app is built. This app was designed according to specifications, it had to look realistic, it had to demonstrate Flutter's technical capabilities, but at the same time it had to be simple enough for students to be able to implement it.

Process

Target users: Udacity students

Feature specifications:

  • List all the items
  • Filtered by TODO
  • Filtered by Done
  • Create/edit/delete items
  • An item has a single title (one-line)
  • An item may have an optional image

I started out by sketching basic functionality and then building an interactive paper prototype in the Marvel POP app. We then tested the prototype with the client.
